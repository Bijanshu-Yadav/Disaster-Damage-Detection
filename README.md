# Disaster-Damage-Detection
Computer Vision Project Detects disaster damage using OpenCV

Automatically detects and classifies structural damage 
by comparing before/after images using Computer Vision.

## 🛠️ Tech Stack
Python, OpenCV, NumPy, Matplotlib

## 📊 Damage Classification
| Level | Damage % |
|-------|----------|
| No Damage | < 5% |
| Minor Damage | 5–15% |
| Major Damage | 15–30% |
| Destroyed | > 30% |
